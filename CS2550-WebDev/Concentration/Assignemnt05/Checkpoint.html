Design Doc

This is the outline of my plan and progress of where I'm at and where I need to go in order to get my game of Concentration completely finished.

Model

Attributes of the Game

  • Timer: Tracks how long the game has gone on
  • Score: Tracks how many matches have been made so far
  • Grid: A yet to be decided grid of cards that all look the same when face down
  • Cards: A yet to be decided number of different cards, each of which needs a match

Timer

I've tested out some basic timer functions, but haven't quite gotten one to work the way I want.
The timer will start when the page loads and end when the game finshes. I would like to have page fade out and display a large congratulations/condolences message with the user's time. It would also be neat to have a leaderboard...

Score

This should be some fairly straightforward logic. What this will do is track how many matches the player has made so far.

The Grid

The grid will probably be a 4x4 table. Easier to program and easier to match the cards. We need to have an even number of cards so that pairs can be made.

Cards

16 cards, and when face down, all look the same
